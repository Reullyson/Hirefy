O sistema resolve o problema da fragmentação de oportunidades ao centralizar vagas de diversas fontes (como Gupy e Indeed) em um único lugar.O grande diferencial está na Inteligência Artificial, que analisa os requisitos da vaga e os dados do usuário para gerar currículos personalizados, aumentando as chances de sucesso em processos seletivos.✨ Funcionalidades PrincipaisAgregador de Vagas: Coleta automatizada de oportunidades via Scraping.Gestão de Perfil 360°: Centralize sua stack, projetos e formação.Gerador de Currículo com IA: Personalização automática baseada na descrição da vaga.Filtros Inteligentes: Busca por tecnologia, localização e nível de senioridade.Exportação: Download de currículos otimizados em formato PDF.🏗️ Arquitetura do SistemaO projeto utiliza uma arquitetura baseada em Microserviços para garantir escalabilidade:Frontend: Aplicação SPA em React para uma experiência fluida.Backend Principal: API em Django responsável pelas regras de negócio.Microserviços (Flask):Scraping: Motores Scrapy e Selenium para coleta de dados.IA: Processamento de linguagem natural para geração de conteúdo.Banco de Dados: MySQL para persistência de dados estruturados.🛠️ TecnologiasLinguagens: Python, JavaScript, SQL.Frameworks: Django, Flask, React.Bibliotecas de Automação: Scrapy, Selenium.Estilização: CSS3 / Tailwind (ou Framework de preferência).👥 EquipeMembroFunçãoIan KilwinyGerente de Projetos / BackendMarcus ViníciusAnalista de Requisitos / BackendReullysonBackend / Frontend / UI/UXIvoFrontendBrunoUI/UX Design / Frontend
